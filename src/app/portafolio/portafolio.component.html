Larksenio HelpDesk Cloud

Aplicación web full stack para gestión de tickets de soporte técnico. Desarrollada con Angular, Node.js, Express, PostgreSQL, Prisma y autenticación JWT. Incluye panel administrador, dashboard con estadísticas, CRUD completo de tickets y despliegue cloud usando Netlify, Render y Neon PostgreSQL. Tecnologías: Angular, TypeScript, Node.js, Express, PostgreSQL, Prisma, JWT, Render, Netlify, Neon. Credenciales demo: Email: admin@larksenio.com Password: Admin12345
